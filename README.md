## Overview
This project implements a Deep Q-Network (DQN) in PyTorch to train an agent to play Atari Pong.

The implementation includes:
- Experience Replay Buffer
- Target Network
- Epsilon-Greedy Exploration
- Convolutional Neural Network for state representation

## Technologies
- Python
- PyTorch
- OpenAI Gym (Atari)
- NumPy
- Matplotlib

## Setup

Install dependencies:

pip install -r requirements.txt

Install Atari ROMs:

AutoROM --accept-license

## Running the Project

Open the notebook:

jupyter notebook dqn_atari_pong.ipynb

Run all cells to begin training.
